# 🌸 Iris Flower Classification

## 📌 Project Overview
This project uses Machine Learning to classify Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

It is built using Python and Scikit-learn.

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas 📊
- NumPy 🔢
- Scikit-learn 🤖
- Matplotlib 📈

---

## 📊 Model Performance
Achieved accuracy: **95%+ (approx)**
